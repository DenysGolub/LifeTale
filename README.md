# LifeTale
Game contains few locations, one battle, 3 mini-quests and end game window.
Game has been developed for PC and Android. 

Genhre of game is expected to be adventure, so in order to finish game you need to walk around and keep your eye on in-game elements! :)

Character Control for PC:

E - interact with NPC

Space - continue next line of dialogue

WASD or Arrows - move the Player

Character Control for Android - need to enable joysticks in hierarchy window.
